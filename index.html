Site Privacy

This site's host, GitHub, may collect some technical information for security and stability reasons. Other than that, this site does not collect or obtain any data from visitors.

This site displays thumbnails from Scratch's content delivery network. See the Scratch Privacy Policy for more information.

Project Privacy

Some projects on Scratch connect to Scratch's Cloud Data service, which allows a limited set of data to be stored or exchanged between users, but only for Scratch users logged into an account with the "Scratcher" status. Learn more about Cloud Data on the Scratch Help Center. When I use Cloud variables to collect data, I follow my own set of privacy practices.

Safety

This site follows the Scratch Community Guidelines, which requires user submissions and interactions within Scratch to be safe and age-appropriate.

Cloud Data can be used to allow for limited communication between users. Forms of free communication over Cloud Data such as writing messages to other users are prohibited by Scratch. Again, you can learn more about Cloud Data on the Scratch Help Center.
